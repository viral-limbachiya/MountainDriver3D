# Mountain Driver 3D

Separate 3D sequel prototype built with Godot 4.7.

## Version 2 features

- Improved SUV handling, grip, braking, suspension and wheel animation
- Three chase-camera modes
- Curving roads, bridge, guardrails, checkpoints and broken asphalt
- Rocks, potholes, mud and road barriers
- Cars and trucks with braking and lane-changing behavior
- Fuel, repair-kit and coin pickups
- Damage, crash effects, dust and checkpoint recovery
- Clear, fog and rain lighting presets
- Missions, rewards, persistent coins and four upgrade categories
- Cinematic menu artwork and mobile touch controls

## Controls

- W / Up: accelerate
- S / Down: brake/reverse
- A / D or Left / Right: steer
- Space: handbrake
- C: change camera
- R: recover at the last checkpoint
- Escape: pause
